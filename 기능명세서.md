9. 관리자 페이지
----------------------------------------------------------------

[관리자 계정 구분]
- 인증 방식 : Google OAuth 로그인
- 관리자 판별 : 로그인 계정이 admin_master 테이블에 등록된 경우 관리자로 인정
- 슈퍼관리자 계정 : go.auth01@gmail.com (고정 지정)

[관리자 권한 구분]

  역할              슈퍼관리자    일반관리자
  ─────────────────────────────────────────
  일반관리자 계정 생성      O         X
  일반관리자 계정 삭제      O         X
  일반관리자 정보 수정      O       본인만
  공지사항 등록/수정/삭제   O         O
  주간공지 파일 등록/삭제   O         O
  학생 정보 등록/수정/삭제  O         O
  시스템 설정 / DB 점검     O         X
  접속 현황 모니터링        O         O

[로그인 보안 정책]
- 로그인 실패 5회 시 계정 자동 잠금 (admin_login_fail_count 기준)
- 잠금 해제 방법 : 아래 두 가지 병행 적용
    ① 자동 해제 : 잠금 후 일정 시간 경과 시 자동 해제 (admin_locked_until 기준)
    ② 수동 해제 : 슈퍼관리자가 관리자 페이지에서 직접 잠금 해제
- 세션 타임아웃 : 30분 무활동 시 자동 로그아웃
- 동시 로그인 : 추후 정책 결정 예정

[접근 보안]
- 관리자 페이지 URL : 추측 불가한 경로 사용 권장
- IP 접근 제한 : 추후 전체적으로 제한 예정 (현재는 전체 허용)
- 2차 인증 : Google 계정 2FA로 대체

[관리자 대시보드 기본 표시 항목]
- 현재 접속 인원 수
- 오늘 등록된 공지 수
- 최근 공지사항 목록 (5건)
- DB / 배포 무료 한도 현황         ← 슈퍼관리자 전용
- 관리자 최근 로그인 이력          ← 슈퍼관리자 전용

[관리 가능한 기능]
- 공지사항 등록/수정/삭제
- 주간 공지 첨부파일 등록/수정/삭제

11. DB 테이블 요구사항 ( 차후 적용 예정 - 연동 컨넥트만 설계 )
----------------------------------------------------------------

[테이블명 : admin_master]
- 용도 : 관리자 계정관리 (계정 별 단일 레코드)
- 컬럼 목록 :

  컬럼명                    타입           설명               제약
  ──────────────────────────────────────────────────────────────────
  admin_is_super_admin      boolean        슈퍼관리자 여부
  admin_id                  text           계정 ID            중복불가
  admin_email               text           이메일             중복불가
  admin_name                text           이름
  admin_passwd_hash         VARCHAR(255)   암호화된 비밀번호
  admin_passwd_changed_at   datetime       비밀번호 변경일
  admin_login_fail_count    smallint       연속 로그인 실패횟수  기본값 0
  admin_locked_until        datetime       잠금 해제 예정 일시   null = 잠금없음
  admin_status              text           계정 상태           active / inactive / locked
  admin_last_login_at       datetime       마지막 로그인 일시
  admin_session_expires_at  datetime       세션 만료 일시
  admin_created_at          datetime       계정 생성일
  admin_created_by          text           생성한 관리자 ID
  admin_updated_at          datetime       최근 수정일
  admin_updated_by          text           수정한 관리자 ID

- 비고 :
  * 비밀번호는 Supabase AES-256 저장 암호화 + pgcrypto bcrypt 해싱 적용
  * 관리자 페이지에서 비밀번호 입력 및 전송 시 암호화 적용
  * 슈퍼관리자(go.auth01@gmail.com)가 일반관리자 최초 등록 후 관리자 페이지 접속 가능
  * 일반관리자는 다른 일반관리자를 등록할 수 없음 (슈퍼관리자 전용)
  * 로그인 실패 5회 시 admin_status = 'locked', admin_locked_until 설정
  * 잠금 해제 : 자동(admin_locked_until 경과) 또는 슈퍼관리자 수동 해제 병행


[테이블명 : user_master]  <- 정보만 제공, 생성은 이후에 진행
- 용도 : 학생별 기본 정보
- 컬럼 목록 :

  컬럼명      타입      설명    제약
  ─────────────────────────────────
  mst_id      char(4)   학번    중복불가
  mst_ban     char(1)   반
  mst_prt     char(1)   그룹
  mst_sex     char(1)   성별
  mst_name    text      이름
  mst_lnce    text      비고

- 참조 관계 : mst_id 참조 key
- 비고 :


[테이블명 : user_penalty]  <- 정보만 제공, 생성은 이후에 진행
- 용도 : 학생별 벌점 내역
- 컬럼 목록 :

  컬럼명          타입       설명      제약
  ──────────────────────────────────────────
  penalty_id      char(4)    학번      중복불가
  penalty_admin   text       담당관리자
  penalty_date    datetime   부여일
  penalty_point   int        벌점
  penalty_part    char(2)    분류
  penalty_code    char(4)    벌점코드
  penalty_cause   text       사유

- 참조 관계 : penalty_id → user_master.mst_id 로 학생 정보 조회
- 비고 : user_penalty 저장 시 user_master 등록 여부 선확인


[테이블명 : user_textnotice]
- 용도 : 전체 공지 사항 관리
- 컬럼 목록 :

  컬럼명                 타입       설명        제약
  ────────────────────────────────────────────────────
  textnotice_id          SERIAL     등록 key    자동생성, 중복불가
  textnotice_date        datetime   등록일시    검색키
  textnotice_owner       text       등록자      검색키
  textnotice_title       text       제목        중복불가, 검색키
  textnotice_body        text       본문
  textnotice_readcnt     int        열람수
  textnotice_hidden      boolean    게시여부

- 비고 :
  * textnotice_hidden : 관리자만 설정 가능, 비공개 시 관리자에게만 노출
  * 구글시트 연동 : 시트에 내용 등록 후 DB 전송 시 자동 등록, 게시 여부 기준으로 자동 처리
  * 본문은 일반 텍스트로 작성 후 셀에 붙여넣는 형태, 안내문 출력창에 단순 표시
